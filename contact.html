Classes

Beginning Knitting. No skills required, just enthusiasm. Make sure to come in early and purchase materials so class can start on me. Class size is limited. Learn the basics of knitting - cast on, bind off, knit, purl, increase and decrease, plus a few tricks. All materials included! Class is three 2-hour sessions. Fee: $55.00.
